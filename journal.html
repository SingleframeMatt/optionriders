Auto-sync with IBKR (Flex Web Service)

  1. In IBKR Client Portal → Performance & Reports → Flex Queries → create a Custom Flex Query. Section: Trades. Format: XML or CSV (both supported).
  2. Include: Symbol, AssetClass, DateTime, TradeDate, Quantity, TradePrice, Proceeds, IBCommission, FifoPnlRealized, Open/CloseIndicator, Buy/Sell.
  3. Enable Flex Web Service under Reports → Settings → Flex Web Service. Generate a token.
  4. Add to your .env: 
    IBKR_FLEX_TOKEN=your_token_here
IBKR_FLEX_QUERY_ID=123456
  5. Click Sync IBKR — or flip the Auto toggle to poll every 15 min while this tab is open.
  6. Or for a one-off: click Import File and upload the CSV/XML you downloaded manually.
